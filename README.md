# المعرض الوطني لدجاج الزينة بالجزائر 2026 — Chicken Expo

موقع متعدد الصفحات (Frontend + Backend) لمعرض دجاج الزينة، من تنظيم **نادي البراهما الجزائري ABC**.

## المميزات
- صفحات: الرئيسية، عن النادي (الأهداف)، الشروط، إنشاء حساب، تسجيل الدخول، مشاركتي، لوحة المدير.
- تسجيل المستخدمين وتسجيل الدخول برقم الهاتف.
- إدخال المشاركة: عدد الطيور، عدد الأقفاص، السلالة، الولاية والبلدية (قوائم منسدلة — 58 ولاية).
- لوحة مدير: إحصائيات + جدول كل المشاركين + تغيير الحالة + حذف الأعضاء.
- قاعدة بيانات PostgreSQL (Neon).

## بيانات المدير
- رقم الهاتف: `0779452212`
- كلمة المرور: `admin123`

## التشغيل

### 1) الواجهة الخلفية (Backend)
```bash
cd backend
npm install
npm run init-db     # ينشئ الجداول ويُهيّئ حساب المدير
npm start           # يعمل على http://localhost:4000
```
ملف `.env` يحتوي على رابط قاعدة بيانات Neon وبيانات المدير.

### 2) الواجهة الأمامية (Frontend)
الخادم يقدّم الواجهة تلقائياً عند فتح: **http://localhost:4000**

أو شغّلها منفصلة بأي خادم ثابت (مثلاً Live Server / `npx serve frontend`)؛
عدّل `frontend/js/config.js` ليشير إلى عنوان الـ backend.

### 3) الشعارات
ضع صور الشعارات في `frontend/assets/` حسب الأسماء في `assets/README.txt`.

## النشر
- Backend: أي مستضيف Node (Render / Railway / VPS). اضبط متغيرات البيئة.
- Frontend: يُقدَّم من نفس الـ backend، أو على Netlify/Vercel مع ضبط `API_BASE`.
